Instructions

Goal of this game is to score as much as possible by hitting the 'moles'
  1. Click the paw with the string "START!" to start the game.
  2. Whack (click) the cat that appears on the 3x3 holes to gain score within the specified duration.
  3. Duration here refers to the the background audio that currently playing, current audio is the a piece called "Cat Dance".

Whack A Cat

Whack A Cat is a simple game simillar to the famous arcade game "Whack A Mole. The main objective for this game is to score as much as possible within the game duration by hitting the 'moles'.

Built using:

Game: HTML, JavaScript, jQuery
Design/Layout: CSS, Twitter Bootstrap, Jack Moore's Colorbox

Concept and Design

Project's Background

At first, I started this project to practice on my general front-end (mostly JavaScript) skills; I thought 'Whack A Mole' is a fun classic arcade that probably could be built using JavaScript. Originally, the game was really simple with placeholder images everywhere. The initial plan was to use normal count down timer as the duration, but I learned that I can play around with HTML5 audio, so I used it for the duration. Eventually. the project grew on me so much that I decided to keep improving it as I learned more stuff.

Concept and Design

Moles Design and Concepts

As stated in the previous section, I wanted it to be unique. But why cat? I was inspired by the card game "Exploding Kitten", and I thought many people likes cat. The original design was really simple with two circle for the eyes, but again, I wanted to make it unique. So I listed a few things that looks good, and distinct like hair, cap, tag, glasses, shade, etc. In the end, I decided to go with shade. But it's not just a normal shade, it's the well know shade that often used in the meme "DEAL WITH IT". This is to give more annoying impression, so that the user is willing to hit it even more.
