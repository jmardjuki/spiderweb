Instructions

Goal of this game is to score as much as possible by hitting the 'moles'
  1. Click the paw with the string "START!" to start the game.
  2. Whack (click) the cat that appears on the 3x3 holes to gain scoren within the specified duration.
  3. Duration here refers to the the background audio that currently playing, current audio piece is Mozart's Twelve Variations on "Ah vous dirai-je, Maman" aka Twinkle Twinkle.

Whack A Cat

Whack A Cat is a simple game simillar to the famous arcade game "Whack A Mole. The main objective for this game is to score as much as possible within the game duration by hitting the 'moles'.

Built using:

Game: HTML, JavaScript, jQuery
Design/Layout/Graphic: CSS, Twitter Bootstrap, Jack Moore's Colorbox
